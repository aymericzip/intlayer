# Пакет vue-intlayer

Пакет `vue-intlayer` надає необхідні інструменти для інтеграції Intlayer у Vue-додатки. Він містить Vue-плагін та composables для роботи з багатомовним контентом.

## Встановлення

```bash
npm install vue-intlayer
```

## Експорти

### Плагін

| Функція           | Опис                                                      |
| ----------------- | --------------------------------------------------------- |
| `installIntlayer` | Vue-плагін для встановлення Intlayer у вашому застосунку. |

### Композиційні функції (Composables)

| Композиційна функція | Опис                                                    |
| -------------------- | ------------------------------------------------------- |
| `useIntlayer`        | Вибирає один словник за його ключем і повертає вміст.   |
| `useDictionary`      | Вибирає один словник за його ключем і повертає вміст.   |
| `useLocale`          | Повертає поточну локаль та функцію для її встановлення. |
| `useIntl`            | Повертає об'єкт Intl для поточної локалі.               |

### Функції

| Функція         | Опис                       |
| --------------- | -------------------------- |
| `getDictionary` | Отримує словник.           |
| `getIntlayer`   | Отримує вміст із словника. |

### Markdown

| Функція                   | Опис                                                               |
| ------------------------- | ------------------------------------------------------------------ |
| `installIntlayerMarkdown` | Плагін Vue для встановлення Intlayer Markdown у вашому застосунку. |
