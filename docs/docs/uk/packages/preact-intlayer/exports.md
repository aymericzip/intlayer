# Пакет preact-intlayer

Пакет `preact-intlayer` надає необхідні інструменти для інтеграції Intlayer у Preact-застосунки. Він включає провайдери та хуки для роботи з багатомовним контентом.

## Встановлення

```bash
npm install preact-intlayer
```

## Експорти

### Провайдер

| Компонент          | Опис                                                                     |
| ------------------ | ------------------------------------------------------------------------ |
| `IntlayerProvider` | Головний провайдер, який обгортає ваш додаток і надає контекст Intlayer. |

### Хуки

| Hook            | Опис                                                                                                          | Пов'язаний документ                                                                                 |
| --------------- | ------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `useIntlayer`   | Базується на `useDictionary`, але інжектує оптимізовану версію словника зі згенерованої декларації.           | -                                                                                                   |
| `useDictionary` | Обробляє об'єкти, що нагадують словники (key, content). Опрацьовує переклади через `t()`, перерахування тощо. | -                                                                                                   |
| `useLocale`     | Повертає поточну локаль та функцію для її встановлення.                                                       | -                                                                                                   |
| `t`             | Вибирає вміст на основі поточної локалі.                                                                      | [переклад](https://github.com/aymericzip/intlayer/blob/main/docs/docs/uk/dictionary/translation.md) |

### Компоненти

| Component          | Description                                              |
| ------------------ | -------------------------------------------------------- |
| `MarkdownProvider` | Провайдер контексту рендерингу Markdown.                 |
| `MarkdownRenderer` | Рендерить Markdown-вміст з користувацькими компонентами. |
