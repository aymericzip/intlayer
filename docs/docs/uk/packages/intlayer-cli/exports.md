# Пакет intlayer-cli

Пакет `intlayer-cli` надає набір команд для керування словниками та конфігурацією Intlayer.

## Встановлення

```bash
npm install intlayer-cli
```

## Експорти

### Команди CLI (Функції)

Пакет експортує функції, які забезпечують роботу команд CLI, дозволяючи запускати операції Intlayer програмно.

Імпорт:

```tsx
import "intlayer-cli";
```

| Функція        | Опис                                                      |
| -------------- | --------------------------------------------------------- |
| `build`        | Будує словники Intlayer.                                  |
| `audit`        | Перевіряє словники на наявність відсутніх перекладів.     |
| `liveSync`     | Синхронізує словники в реальному часі.                    |
| `pull`         | Отримує словники з віддаленого джерела.                   |
| `push`         | Відправляє словники до віддаленого джерела.               |
| `test`         | Запускає тести для словників.                             |
| `transform`    | Перетворює словники між форматами.                        |
| `fill`         | Заповнює словники відсутніми перекладами за допомогою ШІ. |
| `reviewDoc`    | Переглядає документацію для інтернаціоналізації.          |
| `translateDoc` | Перекладає документацію за допомогою AI.                  |
