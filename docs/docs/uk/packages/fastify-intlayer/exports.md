# Пакет fastify-intlayer

Пакет `fastify-intlayer` надає плагін для додатків на Fastify для роботи з інтернаціоналізацією. Він визначає локаль користувача та декорує об'єкт запиту (request).

## Встановлення

```bash
npm install fastify-intlayer
```

## Експорти

### Плагін

Імпорт:

```tsx
import "fastify-intlayer";
```

| Функція    | Опис                                                                                                                                                                                                                                                                                                               | Пов'язана документація                                                                                          |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| `intlayer` | Плагін Fastify, який інтегрує Intlayer у ваш додаток на Fastify. Обробляє визначення локалі зі сховища (cookies, headers), декорує об'єкт запиту даними `intlayer`, що містять `t`, `getIntlayer` та `getDictionary`, а також налаштовує простір імен CLS для програмного доступу протягом життєвого циклу запиту. | [intlayer](https://github.com/aymericzip/intlayer/blob/main/docs/docs/uk/packages/fastify-intlayer/intlayer.md) |

### Функції

Імпорт:

```tsx
import "fastify-intlayer";
```

| Функція         | Опис                                                                                                                                                                                                                                          | Пов'язаний документ                                                                                 |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| `t`             | Глобальна функція перекладу, яка отримує контент для поточної локалі у Fastify. Використовує CLS (Async Local Storage) і повинна використовуватися в контексті запиту, керованому плагіном `intlayer`. Також доступна через `req.intlayer.t`. | [переклад](https://github.com/aymericzip/intlayer/blob/main/docs/docs/uk/dictionary/translation.md) |
| `getIntlayer`   | Отримує словник за його ключем із згенерованої декларації та повертає його вміст для вказаної локалі. Оптимізована версія `getDictionary`. Використовує CLS для доступу до контексту запиту. Також доступна через `req.intlayer.getIntlayer`. | -                                                                                                   |
| `getDictionary` | Обробляє об'єкти словників і повертає вміст для вказаної локалі. Обробляє переклади `t()`, перерахування, Markdown, HTML тощо. Використовує CLS для доступу до контексту запиту. Доступний також через `req.intlayer.getDictionary`.          | -                                                                                                   |
