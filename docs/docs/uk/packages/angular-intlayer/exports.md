# Пакет angular-intlayer

Пакет `angular-intlayer` надає необхідні інструменти для інтеграції Intlayer в Angular-застосунки. Він включає провайдери та сервіси для роботи з багатомовним контентом.

## Встановлення

```bash
npm install angular-intlayer
```

## Експорти

Імпорт:

```tsx
import "angular-intlayer";
```

### Налаштування

| Функція           | Опис                                                     |
| ----------------- | -------------------------------------------------------- |
| `provideIntlayer` | Функція, яка реєструє Intlayer у вашому Angular-додатку. |

### Хуки

| Хук                    | Опис                                                                                                             | Пов'язаний документ |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------- | ------------------- |
| `useIntlayer`          | На основі `useDictionary`, але інжектує оптимізовану версію словника з згенерованої декларації.                  | -                   |
| `useDictionary`        | Обробляє об'єкти, що схожі на словники (ключ, вміст). Обробляє переклади `t()`, перелічення (enumerations) тощо. | -                   |
| `useDictionaryAsync`   | Те саме, що `useDictionary`, але працює з асинхронними словниками.                                               | -                   |
| `useDictionaryDynamic` | Те саме, що `useDictionary`, але працює з динамічними словниками.                                                | -                   |
| `useLocale`            | Повертає поточну локаль та функцію для її встановлення.                                                          | -                   |
| `useIntl`              | Повертає об'єкт Intl для поточної локалі.                                                                        | -                   |
| `useLoadDynamic`       | Хук для завантаження динамічних словників.                                                                       | -                   |

### Компоненти

| Компонент                   | Опис                                                         |
| --------------------------- | ------------------------------------------------------------ |
| `IntlayerMarkdownComponent` | Angular-компонент, який відображає вміст у форматі Markdown. |
