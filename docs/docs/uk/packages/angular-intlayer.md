# Пакет angular-intlayer

Пакет `angular-intlayer` надає необхідні інструменти для інтеграції Intlayer в Angular-застосунки. Він включає провайдери та сервіси для роботи з багатомовним вмістом.

## Встановлення

```bash
npm install angular-intlayer
```

## Експорти

### Налаштування

| Функція           | Опис                                                      |
| ----------------- | --------------------------------------------------------- |
| `provideIntlayer` | Функція для надання Intlayer у вашому Angular-застосунку. |

### Сервіси

| Сервіс            | Опис                                                               |
| ----------------- | ------------------------------------------------------------------ |
| `IntlayerService` | Сервіс, який вибирає словник за його ключем і повертає його вміст. |
| `LocaleService`   | Сервіс, що повертає поточну локаль та функцію для її встановлення. |

### Компоненти

| Компонент                   | Опис                                                         |
| --------------------------- | ------------------------------------------------------------ |
| `IntlayerMarkdownComponent` | Angular-компонент, який відображає вміст у форматі Markdown. |
