# Пакет astro-intlayer

Пакет `astro-intlayer` надає необхідні інструменти для інтеграції Intlayer в додатки Astro. Він налаштовує маршрутизацію на основі локалі та керування словниками.

## Встановлення

```bash
npm install astro-intlayer
```

## Експорти

### Інтеграція

| Функція    | Опис                                                            |
| ---------- | --------------------------------------------------------------- |
| `intlayer` | Інтеграція для Astro, яка налаштовує Intlayer у вашому проєкті. |
