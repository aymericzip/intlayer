# Навички агента

## Налаштування

### Використання CLI

Команда `intlayer init skills` — найпростіший спосіб налаштувати навички агента у вашому проєкті. Вона визначає ваше середовище й встановлює необхідні конфігураційні файли для обраних платформ.

```bash
npx intlayer init skills
```

### Використання Vercel Skill SDK

```bash
npx skills add aymericzip/intlayer-skills
```

### Використання розширення VS Code

1. Відкрийте палітру команд (Ctrl+Shift+P або Cmd+Shift+P).
2. Введіть `Intlayer: Setup AI Agent Skills`
3. Виберіть платформу, яку ви використовуєте (наприклад, `VS Code`, `Cursor`, `Windsurf`, `OpenCode`, `Claude Code`, `GitHub Copilot Workspace` тощо).
4. Виберіть навички, які ви хочете встановити (наприклад, `Next.js`, `React`, `Vite`, `Compiler`, `Configuration`).
5. Натисніть Enter.

## Список навичок

**intlayer-config**

- Наділяє агента здатністю розуміти специфічні налаштування i18n вашого проєкту, дозволяючи йому точно налаштовувати локалі, шаблони маршрутизації та стратегії відкату.

**intlayer-cli**

- Дозволяє агенту автономно керувати життєвим циклом перекладу, включаючи аудит відсутніх перекладів, створення словників та синхронізацію контенту через командний рядок.

**intlayer-angular**

- Дає агенту знання, специфічні для фреймворку, для правильної реалізації реактивних патернів i18n та сигналів відповідно до найкращих практик Angular.

**intlayer-astro**

- Надає агенту знання, необхідні для обробки серверних перекладів та локалізованих патернів маршрутизації, унікальних для екосистеми Astro.

**intlayer-content**

- Навчає агента використанню просунутих контентних вузлів, таких як множина, умови та markdown, для створення багатих, динамічних та локалізованих словників.

**intlayer-next-js**

- Дає агенту глибокі знання для впровадження i18n у серверних та клієнтських компонентах Next.js, забезпечуючи SEO-оптимізацію та безшовну локалізовану маршрутизацію.

**intlayer-react**

- Надає агенту спеціалізовані знання для ефективного впровадження декларативних i18n-компонентів та хуків у будь-якому середовищі на базі React.

**intlayer-preact**

- Оптимізує здатність агента впроваджувати i18n для Preact, дозволяючи йому писати легковажні локалізовані компоненти з використанням сигналів та ефективних реактивних патернів.

**intlayer-solid**

- Дозволяє агенту використовувати дрібнозернисту реактивність SolidJS для високопродуктивного керування локалізованим контентом.

**intlayer-svelte**

- Навчає агента використанню сторів Svelte та ідіоматичного синтаксису для реактивного та типізованого локалізованого контенту в додатках Svelte та SvelteKit.

**intlayer-cms**

- Дозволяє агенту інтегрувати та керувати віддаленим контентом, забезпечуючи живу синхронізацію та робочі проекти віддаленого перекладу через Intlayer CMS.

**intlayer-usage**

- Стандартизує підхід агента до структури проєкту та оголошення контенту, забезпечуючи дотримання найбільш ефективних робочих процесів для вашого i18n-проєкту.

**intlayer-vue**

- Забезпечує агента патернами, специфічними для Vue, включаючи Composables та підтримку Nuxt, для створення сучасних локалізованих вебдодатків.

**intlayer-compiler**

- Спрощує робочий процес агента завдяки автоматичному вилученню контенту, дозволяючи йому писати перекладні рядки прямо в коді без ручного створення файлів словників.
