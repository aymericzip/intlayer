# astro-intlayer 包

`astro-intlayer` 包为将 Intlayer 集成到 Astro 应用中提供必要工具。它配置基于语言/区域的路由和词典管理。

## 安装

```bash
npm install astro-intlayer
```

## 导出

### Integration

| 函数       | 描述                                      |
| ---------- | ----------------------------------------- |
| `intlayer` | 在你的项目中设置 Intlayer 的 Astro 集成。 |
