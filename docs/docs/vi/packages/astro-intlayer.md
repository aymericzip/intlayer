# Gói astro-intlayer

Gói `astro-intlayer` cung cấp các công cụ cần thiết để tích hợp Intlayer vào ứng dụng Astro. Nó cấu hình định tuyến theo locale và quản lý từ điển.

## Cài đặt

```bash
npm install astro-intlayer
```

## Các export

### Tích hợp

| Hàm        | Mô tả                                                  |
| ---------- | ------------------------------------------------------ |
| `intlayer` | Tích hợp Astro thiết lập Intlayer trong dự án của bạn. |
