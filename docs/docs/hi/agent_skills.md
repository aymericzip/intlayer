# एजेंट स्किल्स

## सेटअप

### CLI का उपयोग करके

`intlayer init skills` कमांड आपके प्रोजेक्ट में एजेंट स्किल्स सेटअप करने का सबसे आसान तरीका है। यह आपके वातावरण का पता लगाता है और आपके पसंदीदा प्लेटफ़ॉर्म के लिए आवश्यक कॉन्फ़िगरेशन फ़ाइलें इंस्टॉल करता है।

```bash
npx intlayer init skills
```

### Vercel Skill SDK का उपयोग करके

```bash
npx skills add aymericzip/intlayer-skills
```

### VS Code एक्सटेंशन का उपयोग करके

1. कमांड पैलेट खोलें (Ctrl+Shift+P या Cmd+Shift+P)।
2. `Intlayer: Setup AI Agent Skills` टाइप करें।
3. उस प्लेटफ़ॉर्म को चुनें जिसका आप उपयोग करते हैं (उदा. `VS Code`, `Cursor`, `Windsurf`, `OpenCode`, `Claude Code`, `GitHub Copilot Workspace`, आदि)।
4. वे स्किल्स चुनें जिन्हें आप इंस्टॉल करना चाहते हैं (उदा. `Next.js`, `React`, `Vite`, `Compiler`, `Configuration`)।
5. Enter दबाएं।

## स्किल्स सूची

**intlayer-config**

- एजेंट को आपके प्रोजेक्ट की विशिष्ट i18n सेटिंग्स को समझने का अधिकार देता है, जिससे वह सटीक रूप से लोकेल, राउटिंग पैटर्न और फ़ॉलबैक रणनीतियों को कॉन्फ़िगर कर सके।

**intlayer-cli**

- एजेंट को आपके अनुवाद जीवनचक्र को स्वायत्त रूप से प्रबंधित करने में सक्षम बनाता है, जिसमें कमांड लाइन के माध्यम से लापता अनुवादों का ऑडिट करना, शब्दकोश बनाना और सामग्री सिंक करना शामिल है।

**intlayer-angular**

- कोणीय सर्वोत्तम प्रथाओं के अनुसार प्रतिक्रियाशील i18n पैटर्न और संकेतों को सही ढंग से लागू करने के लिए एजेंट को फ्रेमवर्क-विशिष्ट विशेषज्ञता से लैस करता है।

**intlayer-astro**

- एजेंट को सर्वर-साइड अनुवाद और एस्ट्रो इकोसिस्टम के लिए अद्वितीय स्थानीयकृत राउटिंग पैटर्न को संभालने के लिए ज्ञान प्रदान करता है।

**intlayer-content**

- एजेंट को सिखाता है कि समृद्ध, गतिशील और स्थानीयकृत शब्दकोशों का निर्माण करने के लिए उन्नत सामग्री नोड्स (जैसे बहुवचन, शर्तें और मार्कडाउन) का उपयोग कैसे करें।

**intlayer-next-js**

- एजेंट को Next.js सर्वर और क्लाइंट घटकों में i18n को प्रभावी ढंग से लागू करने की गहराई देता है, जिससे SEO अनुकूलन और निर्बाध स्थानीयकृत राउटिंग सुनिश्चित होती है।

**intlayer-react**

- एजेंट को किसी भी React-आधारित वातावरण के भीतर घोषणात्मक i18n घटकों और हुक को कुशलतापूर्वक लागू करने के लिए विशेष ज्ञान प्रदान करता है।

**intlayer-preact**

- प्रीैक्ट के लिए i18n को लागू करने की एजेंट की क्षमता को अनुकूलित करता है, जिससे वह संकेतों और कुशल प्रतिक्रियाशील पैटर्न का उपयोग करके हल्के, स्थानीयकृत घटक लिख सके।

**intlayer-solid**

- एजेंट को उच्च-प्रदर्शन, स्थानीयकृत सामग्री प्रबंधन के लिए SolidJS की सूक्ष्म-स्तर की प्रतिक्रियाशीलता का लाभ उठाने में सक्षम बनाता है।

**intlayer-svelte**

- एजेंट को Svelte और SvelteKit ऐप्स में प्रतिक्रियाशील और प्रकार-सुरक्षित स्थानीयकृत सामग्री के लिए Svelte स्टोर और मुहावरेदार सिंटैक्स का उपयोग करना सिखाता है।

**intlayer-cms**

- एजेंट को दूरस्थ सामग्री को एकीकृत और प्रबंधित करने की अनुमति देता है, जिससे वह Intlayer CMS के माध्यम से लाइव-सिंकिंग और दूरस्थ अनुवाद वर्कफ़्लो को संभालने में सक्षम हो सके।

**intlayer-usage**

- प्रोजेक्ट संरचना और सामग्री घोषणा के लिए एजेंट के दृष्टिकोण को मानकीकृत करता है, यह सुनिश्चित करते हुए कि यह आपके i18n प्रोजेक्ट के लिए सबसे कुशल वर्कफ़्लो का पालन करता है।

**intlayer-vue**

- आधुनिक, स्थानीयकृत वेब एप्लिकेशन बनाने के लिए एजेंट को Vue-विशिष्ट पैटर्न (कंपोज़ेबल्स और Nuxt समर्थन सहित) से लैस करता है।

**intlayer-compiler**

- स्वचालित सामग्री निष्कर्षण को सक्षम करके एजेंट के वर्कफ़्लो को सरल बनाता है, जिससे वह मैन्युअल शब्दकोश फ़ाइलों के बिना सीधे आपके कोड में अनुवाद योग्य स्ट्रिंग लिख सके।
