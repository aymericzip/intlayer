# intlayer-cli पैकेज

`intlayer-cli` पैकेज Intlayer शब्दकोशों और कॉन्फ़िगरेशन को प्रबंधित करने के लिए कमांड का एक सेट प्रदान करता है।

## इंस्टॉलेशन

```bash
npm install intlayer-cli
```

## एक्सपोर्ट्स

### CLI कमांड्स (फ़ंक्शंस)

पैकेज उन फ़ंक्शंस को एक्सपोर्ट करता है जो CLI कमांड्स को संचालित करते हैं, जिससे आप प्रोग्रामेटिक रूप से Intlayer ऑपरेशन्स को ट्रिगर कर सकते हैं।

Import:

```tsx
import "intlayer-cli";
```

| फ़ंक्शन        | विवरण                                                 |
| -------------- | ----------------------------------------------------- |
| `build`        | Intlayer शब्दकोश बनाता है।                            |
| `audit`        | गायब अनुवादों के लिए शब्दकोशों का ऑडिट करता है।       |
| `liveSync`     | शब्दकोशों को रीयल-टाइम में सिंक करता है।              |
| `pull`         | रिमोट स्रोत से शब्दकोश खींचता है।                     |
| `push`         | शब्दकोशों को रिमोट स्रोत पर पुश करता है।              |
| `test`         | शब्दकोशों पर टेस्ट चलाता है।                          |
| `transform`    | विभिन्न फॉर्मैट के बीच शब्दकोश परिवर्तित करता है।     |
| `fill`         | AI का उपयोग करके गायब अनुवादों से शब्दकोश भरता है।    |
| `reviewDoc`    | अंतरराष्ट्रीयकरण के लिए दस्तावेज़ की समीक्षा करता है। |
| `translateDoc` | AI का उपयोग करके दस्तावेज़ का अनुवाद करता है।         |
