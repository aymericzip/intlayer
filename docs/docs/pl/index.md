# Dokumentacja Intlayer

Witamy w oficjalnej dokumentacji **Intlayer**! Tutaj znajdziesz wszystko, czego potrzebujesz, aby zintegrować, skonfigurować i opanować Intlayer do wszystkich swoich potrzeb związanych z internacjonalizacją (i18n), niezależnie od tego, czy pracujesz z **Next.js**, **React**, **Vite**, **Express** czy innym środowiskiem JavaScript.

Intlayer oferuje elastyczne, nowoczesne podejście do tłumaczenia Twojej aplikacji. Nasza dokumentacja poprowadzi Cię od instalacji i konfiguracji, aż po zaawansowane funkcje, takie jak **tłumaczenie wspomagane AI**, definicje **TypeScript** oraz wsparcie dla **komponentów serwerowych**, umożliwiając stworzenie płynnego, wielojęzycznego doświadczenia.
