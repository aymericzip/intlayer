# مهارات الوكيل

## الإعداد

### استخدام CLI

أمر `intlayer init skills` هو أسهل طريقة لإعداد مهارات الوكلاء في مشروعك. يقوم باكتشاف بيئتك وتثبيت ملفات التهيئة اللازمة للمنصات التي تفضّلها.

```bash
npx intlayer init skills
```

### استخدام Vercel Skill SDK

```bash
npx skills add aymericzip/intlayer-skills
```

### استخدام إضافة VS Code

1. افتح لوحة الأوامر (Ctrl+Shift+P أو Cmd+Shift+P).
2. اكتب `Intlayer: Setup AI Agent Skills`
3. اختر المنصة التي تستخدمها (مثل `VS Code` و `Cursor` و `Windsurf` و `OpenCode` و `Claude Code` و `GitHub Copilot Workspace` إلخ).
4. اختر المهارات التي تريد تثبيتها (مثل `Next.js` و `React` و `Vite` و `Compiler` و `Configuration`).
5. اضغط على Enter.

## قائمة المهارات

**intlayer-config**

- يُمكّن الوكيل من فهم إعدادات i18n الخاصة بمشروعك، مما يسمح له بتكوين اللغات وأنماط التوجيه واستراتيجيات التراجع بدقة.

**intlayer-cli**

- يسمح للوكيل بإدارة دورة حياة الترجمة بشكل مستقل، بما في ذلك تدقيق الترجمات المفقودة، وبناء القواميس، ومزامنة المحتوى عبر سطر الأوامر.

**intlayer-angular**

- يزود الوكيل بخبرة محددة في إطار العمل لتنفيذ أنماط i18n التفاعلية والإشارات بشكل صحيح وفقًا لأفضل ممارسات Angular.

**intlayer-astro**

- يوفر للوكيل المعرفة للتعامل مع الترجمات من جانب الخادم وأنماط التوجيه المحلية الفريدة في نظام Astro البيئي.

**intlayer-content**

- يعلم الوكيل كيفية استخدام عقد المحتوى المتقدمة - مثل الجمع والشروط و markdown - لإنشاء قواميس غنية وديناميكية ومحلية.

**intlayer-next-js**

- يمنح الوكيل العمق اللازم لتنفيذ i18n عبر مكونات Next.js من جانب الخادم والعميل، مما يضمن تحسين محركات البحث والتوجيه المحلي السلس.

**intlayer-react**

- يوفر معرفة متخصصة للوكيل لتنفيذ مكونات وخطافات i18n التصريحية بكفاءة داخل أي بيئة تعتمد على React.

**intlayer-preact**

- يحسن قدرة الوكيل على تنفيذ i18n لـ Preact، مما يسمح له بكتابة مكونات خفيفة ومحلية باستخدام الإشارات والأنماط التفاعلية الفعالة.

**intlayer-solid**

- يُمكّن الوكيل من الاستفادة من التفاعل الدقيق لـ SolidJS لإدارة المحتوى المحلي عالي الأداء.

**intlayer-svelte**

- يعلم الوكيل كيفية استخدام مخازن Svelte والصيغة الاصطلاحية للمحتوى المحلي التفاعلي والآمن برمجياً عبر تطبيقات Svelte و SvelteKit.

**intlayer-cms**

- يسمح للوكيل بدمج وإدارة المحتوى عن بُعد، مما يُمكّنه من التعامل مع سير عمل المزامنة المباشرة والترجمة عن بُعد عبر Intlayer CMS.

**intlayer-usage**

- يوحد نهج الوكيل في هيكل المشروع وإعلان المحتوى، مما يضمن اتباعه لأكثر تدفقات العمل كفاءة لمشروع i18n الخاص بك.

**intlayer-vue**

- يزود الوكيل بأنماط محددة في Vue - بما في ذلك Composables ودعم Nuxt - لبناء تطبيقات ويب حديثة ومحلية.

**intlayer-compiler**

- يبسط سير عمل الوكيل من خلال تمكين استخراج المحتوى التلقائي، مما يسمح له بكتابة سلاسل قابلة للترجمة مباشرة في التعليمات البرمجية الخاصة بك دون ملفات قاموس يدوية.
