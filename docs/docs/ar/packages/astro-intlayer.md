# حزمة astro-intlayer

توفّر حزمة `astro-intlayer` الأدوات اللازمة لدمج Intlayer في تطبيقات Astro. تقوم بإعداد التوجيه حسب اللغة وإدارة القواميس.

## التثبيت

```bash
npm install astro-intlayer
```

## الصادرات

### التكامل

| Function   | Description                                      |
| ---------- | ------------------------------------------------ |
| `intlayer` | تكامل Astro الذي يقوم بإعداد Intlayer في مشروعك. |
