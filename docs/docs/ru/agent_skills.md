# Навыки агента

## Команда `intlayer init skills`

Команда `intlayer init skills`, самый простой способ настроить навыки агента в вашем проекте. Она обнаруживает вашу среду и устанавливает необходимые конфигурационные файлы для предпочитаемых платформ.

```bash
npx intlayer init skills
```

Или с помощью Vercel Skill SDK

```bash
npx skills add aymericzip/intlayer-skills
```

Когда вы запускаете эту команду, она:

1.  Обнаружит фреймворк, который вы используете (например, Next.js, React, Vite).
2.  Спросит, для каких платформ вы хотите установить skills (Cursor, Windsurf, VS Code, OpenCode, Claude Code, GitHub Copilot Workspace и т.д.).
3.  Сгенерирует необходимые файлы конфигурации (например, `.cursor/skills/intlayer-next-js/SKILL.md`, `.windsurf/skills/intlayer-next-js/SKILL.md`, `.opencode/skills/intlayer-next-js/SKILL.md`, `.vscode/mcp.json` и т. д.).

## Поддерживаемые платформы

Intlayer предоставляет документацию для конкретных фреймворков (настройка, использование, метаданные, карта сайта, серверные действия и т. д.), чтобы помочь AI-агенту понять, как работать с Intlayer в вашем конкретном проекте. Эти навыки предназначены для того, чтобы направлять агента через сложности интернационализации, обеспечивая соблюдение правильных шаблонов и лучших практик.

Intlayer поддерживает интеграцию со следующими платформами:

### 1. Cursor

Cursor поддерживает MCP (Model Context Protocol) серверы и пользовательские навыки (custom skills). Выполнение `intlayer init skills`:

- Создаст файл `.cursor/mcp.json` для взаимодействия с MCP-сервером Intlayer.
- Установит навыки для конкретного фреймворка в директорию `.cursor/skills`.

### 2. Windsurf

Windsurf, это AI-IDE. Выполнение `intlayer init skills` установит навыки для конкретного фреймворка в директорию `.windsurf/skills`.

### 3. VS Code

Для пользователей VS Code, особенно тех, кто использует GitHub Copilot или другие расширения, совместимые с MCP, команда:

- Создает конфигурацию `.vscode/mcp.json`.
- Устанавливает навыки для конкретного фреймворка в директорию `skills/` в корне вашего проекта.

### 4. OpenCode

OpenCode, интерактивный CLI-агент, предназначенный для задач разработки программного обеспечения. Intlayer предоставляет специальные навыки, которые помогают OpenCode выполнять задачи по интернационализации. Они устанавливаются в директорию `.opencode/skills`.

### 5. Claude Code

Claude Code можно настроить на использование навыков Intlayer. Команда устанавливает навыки для конкретного фреймворка в директорию `.claude/skills`.

### 6. GitHub Copilot Workspace

GitHub Copilot Workspace позволяет определять пользовательские навыки. Команда устанавливает навыки для конкретного фреймворка в директорию `.github/skills`.
