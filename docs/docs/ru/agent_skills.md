## Команда `intlayer init skills`

Команда `intlayer init skills` — самый простой способ настроить навыки агента в вашем проекте. Она обнаруживает вашу среду и устанавливает необходимые конфигурационные файлы для предпочитаемых платформ.

```bash
npx intlayer init skills
```

Когда вы запускаете эту команду, она:

1.  Обнаружит фреймворк, который вы используете (например, Next.js, React, Vite).
2.  Спросит, для каких платформ вы хотите установить skills (Cursor, VS Code, OpenCode, Claude Code и т.д.).
3.  Сгенерирует необходимые файлы конфигурации (например, `.cursor/mcp.json`, `.vscode/mcp.json` или `.intlayer/skills/*.md`).

## Поддерживаемые платформы

Intlayer поддерживает интеграцию со следующими платформами:

### 1. Cursor

Cursor поддерживает MCP (Model Context Protocol) серверы. Выполнение `intlayer init skills` создаст файл `.cursor/mcp.json`, который позволяет Cursor взаимодействовать с MCP-сервером Intlayer.

### 2. VS Code

Для пользователей VS Code, особенно тех, кто использует GitHub Copilot или другие расширения, совместимые с MCP, команда создаст конфигурацию `.vscode/mcp.json`.

### 3. OpenCode

OpenCode — интерактивный CLI-агент, предназначенный для задач разработки программного обеспечения. Intlayer предоставляет специальные skills, которые помогают OpenCode выполнять задачи по интернационализации.

### 4. Claude Code

Claude Code можно настроить на использование Intlayer skills, добавив сгенерированные файлы конфигурации в его десктопные настройки или в CLI-настройки.
