# Навыки агента

## Настройка

### Использование CLI

Команда `intlayer init skills` — самый простой способ настроить навыки агента в вашем проекте. Она обнаруживает вашу среду и устанавливает необходимые конфигурационные файлы для предпочитаемых платформ.

```bash
npx intlayer init skills
```

### Использование Vercel Skill SDK

```bash
npx skills add aymericzip/intlayer-skills
```

### Использование расширения VS Code

1. Откройте палитру команд (Ctrl+Shift+P или Cmd+Shift+P).
2. Введите `Intlayer: Setup AI Agent Skills`
3. Выберите платформу, которую вы используете (например, `VS Code`, `Cursor`, `Windsurf`, `OpenCode`, `Claude Code`, `GitHub Copilot Workspace` и т. д.).
4. Выберите навыки, которые вы хотите установить (например, `Next.js`, `React`, `Vite`, `Compiler`, `Configuration`).
5. Нажмите Enter.

## Список навыков

**intlayer-config**

- Наделяет агента способностью понимать специфические настройки i18n вашего проекта, позволяя ему точно настраивать локали, шаблоны маршрутизации и стратегии отката.

**intlayer-cli**

- Позволяет агенту автономно управлять жизненным циклом перевода, включая аудит отсутствующих переводов, создание словарей и синхронизацию контента через командную строку.

**intlayer-angular**

- Дает агенту знания, специфичные для фреймворка, для правильной реализации реактивных паттернов i18n и сигналов в соответствии с лучшими практиками Angular.

**intlayer-astro**

- Предоставляет агенту знания, необходимые для обработки серверных переводов и локализованных паттернов маршрутизации, уникальных для экосистемы Astro.

**intlayer-content**

- Обучает агента использованию продвинутых контентных узлов, таких как множественное число, условия и markdown, для создания богатых, динамичных и локализованных словарей.

**intlayer-next-js**

- Дает агенту глубокие знания для внедрения i18n в серверных и клиентских компонентах Next.js, обеспечивая SEO-оптимизацию и бесшовную локализованную маршрутизацию.

**intlayer-react**

- Предоставляет агенту специализированные знания для эффективного внедрения декларативных i18n-компонентов и хуков в любой среде на базе React.

**intlayer-preact**

- Оптимизирует способность агента внедрять i18n для Preact, позволяя ему писать легковесные локализованные компоненты с использованием сигналов и эффективных реактивных паттернов.

**intlayer-solid**

- Позволяет агенту использовать мелкозернистую реактивность SolidJS для высокопроизводительного управления локализованным контентом.

**intlayer-svelte**

- Обучает агента использованию сторов Svelte и идиоматического синтаксиса для реактивного и типизированного локализованного контента в приложениях Svelte и SvelteKit.

**intlayer-cms**

- Позволяет агенту интегрировать и управлять удаленным контентом, обеспечивая живую синхронизацию и рабочие процессы удаленного перевода через Intlayer CMS.

**intlayer-usage**

- Стандартизирует подход агента к структуре проекта и объявлению контента, обеспечивая следование наиболее эффективным рабочим процессам для вашего i18n-проекта.

**intlayer-vue**

- Снабжает агента паттернами, специфичными для Vue, включая Composables и поддержку Nuxt, для создания современных локализованных веб-приложений.

**intlayer-compiler**

- Упрощает рабочий процесс агента за счет автоматического извлечения контента, позволяя ему писать переводимые строки прямо в коде без ручного создания файлов словарей.
