# Навыки агента

## Команда `intlayer init skills`

Команда `intlayer init skills`, самый простой способ настроить навыки агента в вашем проекте. Она обнаруживает вашу среду и устанавливает необходимые конфигурационные файлы для предпочитаемых платформ.

```bash
npx intlayer init skills
```

Или с помощью Vercel Skill SDK

```bash
npx skills add aymericzip/intlayer-skills
```

Когда вы запускаете эту команду, она:

1.  Обнаружит фреймворк, который вы используете (например, Next.js, React, Vite).
2.  Спросит, для каких платформ вы хотите установить skills (Cursor, Windsurf, VS Code, OpenCode, Claude Code, GitHub Copilot Workspace и т.д.).
3.  Сгенерирует необходимые файлы конфигурации (например, `.cursor/skills/intlayer-next-js/SKILL.md`, `.windsurf/skills/intlayer-next-js/SKILL.md`, `.opencode/skills/intlayer-next-js/SKILL.md`, `.vscode/mcp.json` и т. д.).

## Поддерживаемые платформы

Intlayer предоставляет документацию для конкретных фреймворков (настройка, использование, метаданные, карта сайта, серверные действия и т. д.), чтобы помочь AI-агенту понять, как работать с Intlayer в вашем конкретном проекте. Эти навыки предназначены для того, чтобы направлять агента через сложности интернационализации, обеспечивая соблюдение правильных шаблонов и лучших практик.
