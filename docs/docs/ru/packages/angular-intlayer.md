# Пакет angular-intlayer

Пакет `angular-intlayer` предоставляет необходимые инструменты для интеграции Intlayer в Angular-приложения. Он включает провайдеры и сервисы для работы с многоязычным содержимым.

## Установка

```bash
npm install angular-intlayer
```

## Экспорты

### Настройка

| Функция           | Описание                                                    |
| ----------------- | ----------------------------------------------------------- |
| `provideIntlayer` | Функция для подключения Intlayer в ваше Angular-приложение. |

### Сервисы

| Сервис            | Описание                                                               |
| ----------------- | ---------------------------------------------------------------------- |
| `IntlayerService` | Сервис, который выбирает словарь по его ключу и возвращает содержимое. |
| `LocaleService`   | Сервис, который возвращает текущую локаль и функцию для её установки.  |

### Компоненты

| Компонент                   | Описание                                              |
| --------------------------- | ----------------------------------------------------- |
| `IntlayerMarkdownComponent` | Angular-компонент, который рендерит Markdown-контент. |
