# Пакет express-intlayer

Пакет `express-intlayer` предоставляет middleware для приложений на Express для работы с интернационализацией. Он определяет локаль пользователя и предоставляет функции перевода.

## Установка

```bash
npm install express-intlayer
```

## Экспорты

### Middleware (промежуточное ПО)

Импорт:

```tsx
import "express-intlayer";
```

| Функция    | Описание                                                                                                                                                                                                                                                                                   | Связанный документ                                                                                              |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------- |
| `intlayer` | Express middleware, который определяет локаль пользователя и заполняет `res.locals` данными Intlayer. Выполняет определение локали по cookies/заголовкам, внедряет `t`, `getIntlayer` и `getDictionary` в `res.locals`, и настраивает namespace CLS для доступа в жизненном цикле запроса. | [intlayer](https://github.com/aymericzip/intlayer/blob/main/docs/docs/ru/packages/express-intlayer/intlayer.md) |

### Функции

Импорт:

```tsx
import "express-intlayer";
```

| Функция         | Описание                                                                                                                                                                                                             | Связанный документ                                                                                 |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| `t`             | Функция перевода, которая получает контент для текущей локали. Работает в рамках жизненного цикла запроса, управляемого middleware `intlayer`. Использует CLS (Async Local Storage) для доступа к контексту запроса. | [перевод](https://github.com/aymericzip/intlayer/blob/main/docs/docs/ru/dictionary/translation.md) |
| `getIntlayer`   | Извлекает словарь по его ключу из сгенерированного декларационного файла и возвращает его содержимое для указанной локали. Оптимизированная версия `getDictionary`. Использует CLS для доступа к контексту запроса.  | -                                                                                                  |
| `getDictionary` | Обрабатывает объекты словарей и возвращает содержимое для указанной локали. Обрабатывает переводы `t()`, перечисления, markdown, HTML и т.д. Использует CLS для доступа к контексту запроса.                         | -                                                                                                  |
