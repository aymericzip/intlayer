# Пакет astro-intlayer

Пакет `astro-intlayer` предоставляет необходимые инструменты для интеграции Intlayer в приложения на Astro. Он настраивает маршрутизацию на основе локали и управление словарями.

## Установка

```bash
npm install astro-intlayer
```

## Экспортируемые элементы

### Интеграция

| Функция    | Описание                                                            |
| ---------- | ------------------------------------------------------------------- |
| `intlayer` | Интеграция для Astro, которая настраивает Intlayer в вашем проекте. |
