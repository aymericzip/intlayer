# Пакет intlayer-cli

Пакет `intlayer-cli` предоставляет набор команд для управления словарями Intlayer и конфигурацией.

## Установка

```bash
npm install intlayer-cli
```

## Экспорты

### CLI-команды (функции)

Пакет экспортирует функции, которые обеспечивают работу CLI-команд.

| Функция     | Описание                                                    |
| ----------- | ----------------------------------------------------------- |
| `build`     | Создает словари Intlayer.                                   |
| `audit`     | Проводит аудит словарей на предмет отсутствующих переводов. |
| `liveSync`  | Синхронизирует словари в реальном времени.                  |
| `pull`      | Загружает словари из удаленного источника.                  |
| `push`      | Отправляет словари в удаленный источник.                    |
| `test`      | Запускает тесты для словарей.                               |
| `transform` | Преобразует словари между форматами.                        |
