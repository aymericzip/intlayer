# Пакет next-intlayer

Пакет `next-intlayer` предоставляет необходимые инструменты для интеграции Intlayer в приложения на Next.js. Он поддерживает как App Router, так и Page Router, включая middleware для маршрутизации на основе локали.

## Установка

```bash
npm install next-intlayer
```

## Экспорты

### Промежуточное ПО

| Функция              | Описание                                                                           |
| -------------------- | ---------------------------------------------------------------------------------- |
| `intlayerMiddleware` | Next.js middleware для обработки маршрутизации на основе локали и перенаправлений. |

### Провайдеры

| Компонент                | Описание                                                    |
| ------------------------ | ----------------------------------------------------------- |
| `IntlayerClientProvider` | Провайдер для клиентских компонентов в Next.js.             |
| `IntlayerServerProvider` | Провайдер для серверных компонентов в Next.js (App Router). |

### Хуки (на стороне клиента)

Переэкспортирует большинство хуков из `react-intlayer`.

| Хук             | Описание                                               |
| --------------- | ------------------------------------------------------ |
| `useIntlayer`   | Выбирает словарь по его ключу и возвращает содержимое. |
| `useDictionary` | Выбирает словарь по его ключу и возвращает содержимое. |
| `useLocale`     | Возвращает текущую локаль и функцию для её установки.  |
| `useI18n`       | Возвращает текущие значения контекста Intlayer.        |

### Функции (Server-side)

| Функция                | Описание                                                             |
| ---------------------- | -------------------------------------------------------------------- |
| `t`                    | Серверная версия функции перевода для Next.js App Router.            |
| `generateStaticParams` | Генерирует статические параметры для динамических маршрутов Next.js. |

### Типы

| Тип                  | Описание                                         |
| -------------------- | ------------------------------------------------ |
| `NextPageIntlayer`   | Тип для страниц Next.js с поддержкой Intlayer.   |
| `NextLayoutIntlayer` | Тип для layout'ов Next.js с поддержкой Intlayer. |
