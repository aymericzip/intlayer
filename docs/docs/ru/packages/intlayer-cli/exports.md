# Пакет intlayer-cli

Пакет `intlayer-cli` предоставляет набор команд для управления словарями Intlayer и конфигурацией.

## Установка

```bash
npm install intlayer-cli
```

## Экспорты

### Команды CLI (функции)

Пакет экспортирует функции, лежащие в основе команд CLI, позволяя запускать операции Intlayer программно.

Импорт:

```tsx
import "intlayer-cli";
```

| Функция        | Описание                                                  |
| -------------- | --------------------------------------------------------- |
| `build`        | Собирает словари Intlayer.                                |
| `audit`        | Проверяет словари на предмет отсутствующих переводов.     |
| `liveSync`     | Синхронизирует словари в реальном времени.                |
| `pull`         | Загружает словари из удалённого источника.                |
| `push`         | Отправляет словари в удалённый источник.                  |
| `test`         | Запускает тесты для словарей.                             |
| `transform`    | Преобразует словари между форматами.                      |
| `fill`         | Заполняет словари отсутствующими переводами с помощью ИИ. |
| `reviewDoc`    | Проверяет документацию на предмет интернационализации.    |
| `translateDoc` | Переводит документацию с помощью AI.                      |
