# Пакет vue-intlayer

Пакет `vue-intlayer` предоставляет необходимые инструменты для интеграции Intlayer в приложения на Vue. Он включает Vue-плагин и composables для работы с многоязычным контентом.

## Установка

```bash
npm install vue-intlayer
```

## Экспорты

### Плагин

| Функция           | Описание                                              |
| ----------------- | ----------------------------------------------------- |
| `installIntlayer` | Плагин Vue для установки Intlayer в вашем приложении. |

### Композиционные функции

| Композиционный хук | Описание                                               |
| ------------------ | ------------------------------------------------------ |
| `useIntlayer`      | Выбирает словарь по его ключу и возвращает содержимое. |
| `useDictionary`    | Выбирает словарь по его ключу и возвращает содержимое. |
| `useLocale`        | Возвращает текущую локаль и функцию для её установки.  |
| `useIntl`          | Возвращает объект Intl для текущей локали.             |

### Функции

| Функция         | Описание                      |
| --------------- | ----------------------------- |
| `getDictionary` | Извлекает словарь.            |
| `getIntlayer`   | Извлекает контент из словаря. |

### Markdown

| Функция                   | Описание                                                       |
| ------------------------- | -------------------------------------------------------------- |
| `installIntlayerMarkdown` | Vue-плагин для установки Intlayer Markdown в вашем приложении. |
