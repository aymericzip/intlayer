## ¿Qué es Intlayer?

La mayoría de las bibliotecas i18n son demasiado complejas, demasiado rígidas o no están diseñadas para marcos modernos.

Intlayer es una **solución i18n moderna** para aplicaciones web y móviles.  
Es independiente del marco, **impulsado por IA**, e incluye un **CMS y editor visual** gratuitos.

Con **archivos de contenido por localidad**, **autocompletado de TypeScript**, **diccionarios tree-shakable** e **integración CI/CD**, Intlayer hace que la internacionalización sea **más rápida, más limpia y más inteligente**.

## 📦 Instalación

Comience su viaje con Intlayer hoy y experimente un enfoque más fluido y potente para la internacionalización.

<a href="https://intlayer.org/doc/concept/content" rel="">
  <img src="https://img.shields.io/badge/Comenzar-FFFFFF?style=for-the-badge&logo=rocket&logoColor=black" />
</a>

```bash packageManager="npm"
npm install intlayer react-intlayer
```

```bash packageManager="yarn"
yarn add intlayer react-intlayer
```

```bash packageManager="pnpm"
pnpm add intlayer react-intlayer
```

```bash packageManager="bun"
bun add intlayer react-intlayer
```

⚡ Inicio rápido (Next.js)

```ts
// intlayer.config.ts
import { Locales, type IntlayerConfig } from "intlayer";

const config: IntlayerConfig = {
  internationalization: {
    locales: [Locales.ENGLISH, Locales.FRENCH, Locales.SPANISH],
    defaultLocale: Locales.ENGLISH,
  },
};

export default config;
```

```ts
// app/home.content.ts
import { t, type Dictionary } from "intlayer";

const content = {
  key: "home",
  content: {
    title: t({
      en: "Home",
      fr: "Accueil",
      es: "Inicio",
    }),
  },
} satisfies Dictionary;

export default content;
```

```tsx
// app/page.tsx
import { useIntlayer } from "react-intlayer";

const HomePage = () => {
  const { title } = useIntlayer("home");

  return <h1>{title}</h1>;
};
```

## Sistema de gestión de contenidos multilingüe

Más que una biblioteca i18n, Intlayer es un **sistema de gestión de contenidos multilingüe** completo. Un CMS completo está disponible de forma gratuita en [app.intlayer.org](https://app.intlayer.org/?utm_source=chatgpt.com).

Intlayer conecta a **desarrolladores**, **redactores** y **agentes de IA** en un único flujo de trabajo para crear y mantener sitios web multilingües sin esfuerzo. Intlayer reemplaza la siguiente pila en una sola solución:

- Soluciones i18n (por ejemplo, `i18next`, `next-intl`, `vue-i18n`)
- TMSs (Sistemas de Gestión de Traducciones) (por ejemplo, Crowdin, Phrase, Lokalise)
- CMSs Headless (por ejemplo, Contentful, Strapi, Sanity)
