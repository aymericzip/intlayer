# astro-intlayer 패키지

`astro-intlayer` 패키지는 Intlayer를 Astro 애플리케이션에 통합하는 데 필요한 도구를 제공합니다. 로케일 기반 라우팅과 딕셔너리 관리 설정을 담당합니다.

## 설치

```bash
npm install astro-intlayer
```

## 내보내기(Exports)

### 통합(Integration)

| 함수       | 설명                                       |
| ---------- | ------------------------------------------ |
| `intlayer` | 프로젝트에 Intlayer를 설정하는 Astro 통합. |
