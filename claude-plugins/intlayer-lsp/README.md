# intlayer-lsp

Intlayer language server for Claude Code — provides **Go-to-Definition** for `useIntlayer`, `getIntlayer`, `useDictionary`, and `getDictionary` calls across all `*-intlayer` packages (`next-intlayer`, `react-intlayer`, `vue-intlayer`, `svelte-intlayer`, and more).

## Supported Extensions

`.ts`, `.tsx`, `.js`, `.jsx`, `.vue`, `.svelte`

## Installation

Install the Intlayer language server globally via npm:

```bash
npm install -g @intlayer/lsp
```

```bash
yarn global add @intlayer/lsp
```

```bash
pnpm add -g @intlayer/lsp
```
